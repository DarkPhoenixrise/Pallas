# 🎬 Project Demo

Watch the full demonstration of **Pallas**, a Retrieval-Augmented Generation (RAG) based LLM designed for accurate, grounded question answering over custom document collections.

[![Watch Demo](https://img.youtube.com/vi/gS9AZuihq2g/maxresdefault.jpg)](https://youtu.be/gS9AZuihq2g)

👉 **[Watch on YouTube](https://youtu.be/gS9AZuihq2g)**

The video demonstrates:
- Document ingestion and automatic knowledge base creation
- Semantic chunking, embedding generation, and ChromaDB vector indexing
- Intelligent retrieval of the most relevant document chunks
- Query expansion and context-aware answer generation using Llama 3
- Grounded responses generated strictly from retrieved context to minimize hallucinations
- End-to-end Retrieval-Augmented Generation (RAG) pipeline running locally
